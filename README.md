# crudvehicles

## Requirements
* Python 3.6
* [pipenv](https://docs.pipenv.org/)

## Environment
* `pipenv --python 3.6`
* `pipenv shell`
* `pip install -r requirements.txt`

* `python manage.py migrate`

## Execute project

* `python manage.py runserver`

* Access [http://localhost:8000/api/](https://docs.pipenv.org/) for list all urls

## Execute test
* `make run-test`
